# Souraphat_Sandy_Tp2 sans lfs

Créé par Sandy Souraphat.

Developpé with Unreal Engine 5.6.1.

## Concept

Mon concept est principalement de créer un jeu de Lock and Key où que le joueur explore la première niveau en essayant de trouver une clé magique auquel il peut ouvrir la mystérieuse porte vers le deuxième niveau. Dans le deuxième niveau, le joueur devra trouver deux différents artefacts magiques qui serviront pour accéder aux autres salles secrets. Dans chaque zones secrets, contenant des obstacles complexes à surmonter avant d'atteindre la salle finale. Une fois que le joueur a toute réussi de trouver les éléments, le joueur pourra ouvrir la porte principale menant au trésor ancien.
